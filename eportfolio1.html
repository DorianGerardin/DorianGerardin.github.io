Premier E-portfolio

par Dorian gerardinFEv 2018

Langages utilisés : HTML / CSS / MATERIALIZE

Ce projet est le premier E-portfolio que j'ai été amené à réaliser dans le cadre du module IHM afin de se familiariser avec le Material Design et le framework CSS de Google : Materialize

Réalisation

Pour ce premier E-portfolio, j'ai voulu lier les choses que j'aimes : la nature, l'astro-physique ainsi que l'informatique. C'est pourquoi on y retrouve une charte graphique tournée dans les verts ainsi que des images de fond correspondant à un ciel étoilé et un fond de code informatique. J'avais intégré dans ce E-portfolio 3 projets qui sont aussi présentés dans celui-ci. J'avais aussi rajouté des éléments personnels comme une carte mentale et un nuage de mot. Ce E-portfolio ne me correspondait plus, et j'ai décidé d'en réaliser un niuveau, qui était plus épuré et plus visuel.
