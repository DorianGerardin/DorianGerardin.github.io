Site de eCommerce

par Dorian gerardin, nathanael capell, van derhoeven paul OCT 2019 - JAN 2020

Langages utilisés : PHP / SQL / HTML / CSS

Ce projet est un site de e-Commerce développé en PHP,  en lien avec une base de données SQL. Ce projet m'a permis d'apprendre le design pattern Model View Controller, ainsi que les nombreuses méchaniques de requêtes vers un serveur notamment avec les session et les cookies. C'est un projet que j'ai vraiment apprécié.

Possibilités

Ici, on vend des items de jeu vidéos, sur une listes de nombreux jeux vidéos. Un utilisateur peut ajouter des articles au panier, puis passer command ensuite s'il le souhaite. Il y a aussi une gestion de comptes utilisateurs. En effet, il faut être connecté pour passer une commande. A chaque commande, l'utilisateur gagne  de l'expérience et peut monter de niveau. Après chaque niveau passé, l'utilisateur aura une réduction sur son prochain achat. Il y aussi une gestion administrateur, où l'on peut gérer la liste de tous les utlisateurs, modifier ou ajouter des jeux/items pour ces jeux, gérer les commandes ou encore supprimer des jeux/items/utilisateurs.
